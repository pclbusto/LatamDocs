LATAM Voucher Class Type

Descripción

Este maestro es una tipificación para las clases de comprobantes. Hay varias secciones en esta entidad, cada una representa un tipo de documento que puede ser usada en distintas entidades por ejemplo el grupo facturas tiene facturas de ventas, factura de compras, factura de servicios. Cada uno de estos check representa que puede ser usada como la extension de un determinado documento.

Sección General

No.

Descripción: Identificador de tipo clase de comprobante.

Tipo:Code

Descripción

Descripción: texto descriptivo.

Tipo:text

Prefijo

Descripción: Cadena de caracteres limitada por el campo long. prefijo tipo clase comprobante, en la configuración de latam

Tipo:boolean
